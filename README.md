## English

Drag2Note-Lite is a lightweight, efficient Windows desktop note-taking application designed for quick information capture via drag-and-drop. It extracts content from files and images into searchable, tagged notes.

### ğŸŒŸ Features
- **Drag & Drop Capture**: Instantly create notes by dragging files or images onto the floating drop zone.
- **Smart Tagging**: Automatically adds date tags and supports custom tag management with drag-to-sort.
- **Portable & Minimalist**: All data is stored locally in the `UserData` folder. No cloud required.
- **Dark Mode Support**: Seamlessly syncs with system themes.
- **Global Hotkeys**: Quick access to the dashboard and floating zone.
- **Lightweight WPF**: Clean UI with a premium feel and smooth animations.

### ğŸš€ Getting Started
1. Download the latest `Drag2Note-Lite_Setup.exe` from the [Releases](https://github.com/lesserafim4ever0502/Drag2Note-Lite/releases) (if available) or build from source.
2. Run the installer and choose your installation path.
3. Use `Alt+Q` (default) to quickly focus the dashboard.
